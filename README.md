# Netflix Data Analysis

This project performs Exploratory Data Analysis (EDA) on a Netflix movie dataset using Python and Google Colab.

## Project Objective

The goal of this project is to analyze Netflix movie data and discover insights related to:
- Movie genres
- Popularity
- Vote averages
- Release trends
- Most popular movies

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## Dataset Information

- Dataset Name: `mymoviedb.csv`
- Total Records: 9827 movies
- Dataset contains movie-related information such as:
  - Genre
  - Popularity
  - Vote Average
  - Release Date
  - Original Language

---

## Questions Answered

1. What is the most frequent genre of movies released on Netflix?
2. Which category has the highest votes in the Vote_Average column?
3. What movie has the highest popularity and what is its genre?
4. What movie has the lowest popularity and what is its genre?
5. Which year has the highest number of movie releases?

---

## Key Insights

- Drama is the most frequent genre in the dataset.
- Spider-Man: No Way Home has the highest popularity.
- Recent years show a higher number of movie releases.
- Popularity and vote distributions were analyzed using visualizations.

---

## Visualizations Included

- Genre Distribution
- Popularity Analysis
- Vote Average Distribution
- Release Year Histogram
- Count Plots and Histograms
